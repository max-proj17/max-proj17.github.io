Infrared Security System

The purpose of the electric eye light beam-interrupter safety system is to design, develop, and test a novel safety system aimed at reliably detecting, and responding to, physical obstructions in a given environment. This project addresses a critical real-world problem wherein accurate and dependable sensor data are essential for ensuring the safety and functionality of various applications. The system consists of a transmitter, which emits a modulated infrared (IR) beam, and a receiver, which detects interruptions in the beam via hardware filtering and triggers an alert. Through a comprehensive exploration of design choices, experimentation, and testing, this report provides insights into the development of a safety system that performs effectively under diverse lighting conditions, while also incorporating an SMS alert feature.

CUDA Accelerated Convolutional Neural Network (CNN)

Currently developing a state-of-the-art Convolutional Neural Network (CNN) leveraging CUDA for accelerated computing. This project involves creating a custom dataset class for the MNIST dataset, integrating it with the powerful LibTorch library. The main focus is on enhancing the performance of neural network training by utilizing the parallel computing capabilities of NVIDIA's CUDA technology. Key features include:

  • Custom MNIST dataset handling with advanced data preprocessing and normalization techniques.
  • Dynamic device selection enabling training on both CPU and GPU, harnessing CUDA's power when available.
  • Implementation of a sequential data loader for efficient batch processing and optimized memory usage.
  • Integration with LibTorch for sophisticated model training, error handling, and performance metrics evaluation.

This project is a testament to applying advanced computing techniques to solve complex problems in machine learning and artificial intelligence.
